December’s UDPL Shooting Event -
(Note:  This IS NOT AN IDPA MATCH. All shooters are invited...)

 

What:

 Handgun and Shotgun (or handgun only) match at the FARM.


When:

December 18th.

  PLEASE NOTE THAT THIS IS THE THIRD SATURDAY!!


Sign in:

8:30 a.m.


Fee:

New Shooters = $10.00

UDPL Member Shooters = $13.00

Non-UDPL Shooters = $15.00

What to Bring:

Handgun and Shotgun, or just handgun (See “Description of Match” below).

 

Divisions:

PISTOL Only = Only a pistol is used for the match

REV Only = Only a revolver is used for the match

P-PUMP = Pistol and PUMP shotgun is used for the match

R-PUMP = Revolver and PUMP shotgun is used for the match

P-AUTO = Pistol and AUTO shotgun is used for the match

R-AUTO = Revolver and AUTO shotgun is used for the match


Minimum Ammo Requirements:

Pistol/Revolver = 60 rounds (or 120 if using Pistol/Revolver only)
Birdshot = 48 rounds


Note that these are ABSOLUTE minimums, so please bring more, unless you
don't plan to miss.

Description of Match:

There are six stages - all Steel - all Vickers.
There is no movement because I don't know if we will have ice and snow to slip on.


The first string will be with pistol/revolver, the second with shotgun. 

However, if a shooter would like to shoot all stages with pistol or revolver only, then their division will be "PISTOL Only" or ”REV Only” and they will simply shoot both strings per stage with pistol or revolver.

 

ALSO, don’t forget the annual New Year’s Day 3-Gun Match. Specifics will be posted prior to that match.
